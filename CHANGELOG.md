# 更新日志

本文件记录了该项目的所有重要更改。

## [1.3.1] - 2024

### 新增
- 完整的中文注释和文档
- 详细的 API 使用说明

### 功能
- 支持 Android 和 iOS 平台的文档扫描
- 自动检测和裁剪文档边缘
- Android 支持多页扫描
- Android 支持从图库导入图像
- iOS 支持 PNG 和 JPEG 输出格式
- iOS 支持可配置的 JPEG 压缩质量

### 技术特性
- 使用 Google ML Kit 文档扫描器（Android）
- 使用 VisionKit 框架（iOS）
- 自动权限请求和处理
- 完整的错误处理

### 系统要求
- Android: API 21+
- iOS: 13.0+
- Flutter: 2.5.0+
- Dart: 3.0.0+
